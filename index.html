The crawled data are continually collected from Twitch every five minutes in a one-month period (Feb. 1st-28th, 2015). Through the official APIs (http://dev.twitch.tv/), our multi-thread crawler obtained information from each broadcaster and the official system dashboard

Twitch crawler: Our multi-thread crawler does not need Twitch's API client-ID and avoids the limitation for the maximum number of objects to return in each request.
